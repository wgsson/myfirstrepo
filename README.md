Brancher inom Git 
Med hjälp av Git kan flera personer arbeta på samma projekt men i olika brancher för att inte störa huvudprojektet. Jag har förstått det som att en branch är en egen kopia av arbetet när ideer kan testas och byggas innan det blir initierat i huvudprojektet (master).

Några exempel som jag lärt mig är: git branch "namn på branch" = skapar en 
ny branch 
git checkout "namn på branch" = byter till den branchen
git checkout -b "namn på branch"= skapa en ny branch och byt till den branchen direkt
